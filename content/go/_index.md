## 📌 章节概览

Go 语言是当前后端面试的核心考点，本章节覆盖从基础语法到底层原理的高频面试题。

| 模块 | 核心考点 |
|------|---------|
| 基础语法 | 切片、Map、接口、make/new、函数传参 |
| 并发编程 | Goroutine、Channel、GMP调度模型 |
| 内存管理 | mspan、mcache、mcentral、内存逃逸 |
| 垃圾回收 | 三色标记法、写屏障、GC触发机制 |
| 同步原语 | Mutex、WaitGroup、sync.Map |
