## 📌 章节概览

MySQL 是后端开发必考项，重点考察对底层原理的理解，而非简单的使用。

| 模块 | 核心考点 |
|------|---------|
| 零、基础原理 | SQL执行流程、存储引擎架构 |
| 一、事务机制 | ACID特性、隔离级别、MVCC多版本控制 |
| 二、日志系统 | redo log重做日志、undo log回滚日志 |
| 三、索引原理 | B+树结构、索引分类、回表与覆盖索引 |
| 四、锁机制 | 行锁表锁、间隙锁、乐观锁与悲观锁 |
