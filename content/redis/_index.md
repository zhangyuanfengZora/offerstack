## 📌 章节概览

Redis 作为缓存中间件，是面试必考内容，重点考察原理和实际应用场景。

| 模块 | 核心考点 |
|------|---------|
| 数据结构 | String/Hash/List/Set/ZSet 底层实现 |
| 持久化 | RDB vs AOF，混合持久化 |
| 高可用 | 主从复制、哨兵、Cluster 集群 |
| 缓存问题 | 缓存击穿、穿透、雪崩 |
| 分布式 | 分布式锁、RedLock |
